# Python While Loop Guessing Game

This project demonstrates the use of while loops, conditional logic, and user input handling in Python through a simple number guessing game.

## Overview
The program continuously prompts the user to guess a secret number until the correct value is entered. During each attempt, the program provides feedback indicating whether the number entered is even or odd.

## Features
- Continuous loop using a while statement
- Conditional logic to determine even vs odd numbers
- User input handling and validation
- Interactive command-line game

## Skills Demonstrated
- Python programming
- While loops
- Conditional statements (if/else)
- User interaction
- Basic control flow

## File
- `guessing_game.py`

## Why This Project Matters
This project highlights foundational programming concepts such as loops and decision-making, which are essential for building more complex applications in data processing, automation, and AI systems.
